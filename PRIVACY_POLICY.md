# B-HUB POS Privacy Policy

**Effective Date:** February 15, 2026  
**Last Updated:** February 15, 2026

---

## 1. Introduction

B-HUB POS ("the App") is a professional Point of Sale system designed for grocery retail businesses in Oman. This Privacy Policy explains how we handle data collected through the App.

**Owner:** BHAEES Trading L.L.C.  
**Contact:** support@bhub.om

---

## 2. Data Ownership

**All transaction data, customer ledgers (Khat), inventory records, and sales analytics belong exclusively to the Store Owner.**

- The Store Owner has full control over their business data
- Data can be exported or deleted at any time upon request
- B-HUB acts solely as a data processor, not a data controller

---

## 3. Data Collection

The App collects the following types of data:

### 3.1 Business Data
- **Sales Transactions:** Product names, quantities, prices, payment methods, timestamps
- **Inventory Records:** Product details, stock levels, pricing
- **Customer Information:** Names, phone numbers (for loyalty program only)
- **Staff Activity:** Login times, shift records, cashier IDs

### 3.2 Technical Data
- **Device Information:** Device model, operating system version (for compatibility)
- **Location Data:** Store location (for VAT-compliant receipt headers only)
- **Usage Analytics:** App performance metrics (crashes, load times)

---

## 4. Data Usage

We use collected data solely for:

1. **Processing sales transactions**
2. **Generating tax-compliant receipts** (Oman Tax Authority requirements)
3. **Inventory management and stock alerts**
4. **Customer loyalty program management**
5. **Owner remote monitoring dashboards**
6. **Technical support and troubleshooting**

**We do NOT:**
- Sell your data to third parties
- Use data for advertising purposes
- Share data with competitors
- Access data without explicit permission

---

## 5. Data Security

### 5.1 Encryption
- **In Transit:** All data transmitted between the App and cloud servers uses HTTPS/TLS encryption
- **At Rest:** Database records are encrypted using industry-standard AES-256 encryption

### 5.2 Access Control
- Role-based authentication (Owner, Manager, Cashier)
- Username + Password required (no PIN-only access)
- Session timeouts for inactive users
- Audit logs for all data access

### 5.3 Cloud Infrastructure
- Data hosted on secure cloud servers (Supabase/Firebase)
- Regular automated backups
- 99.9% uptime SLA
- ISO 27001 certified data centers

---

## 6. Data Retention

In compliance with **Oman Tax Authority (OTA) regulations:**

- **Transaction records:** Retained for **10 years** minimum
- **Tax invoices:** Archived for audit purposes
- **Customer data:** Retained as long as the account is active
- **Staff records:** Retained for employment compliance

Store Owners may request data deletion after the mandatory retention period.

---

## 7. Permissions

The App requests the following device permissions:

### 7.1 Camera Access
- **Purpose:** Zero-lag barcode scanning for products
- **Usage:** Only activated when scanning barcodes in POS screen
- **Storage:** Images are NOT stored or transmitted

### 7.2 Location Access
- **Purpose:** VAT-compliant receipt headers (store location)
- **Usage:** One-time access during initial setup
- **Precision:** City-level only (not GPS coordinates)

### 7.3 Storage Access
- **Purpose:** Offline transaction queue and product caching
- **Usage:** Local database for offline functionality
- **Sync:** Automatically syncs when internet connection restored

### 9.2 Data Protection

- GDPR-inspired data handling practices
- Right to access, rectify, and delete personal data

## 10. User Rights

1. **Access:** Request a copy of all stored data
2. **Rectification:** Correct inaccurate information
3. **Deletion:** Request data deletion (subject to legal retention requirements)
4. **Portability:** Export data in machine-readable format (CSV, JSON)
5. **Objection:** Opt-out of non-essential data processing

To exercise these rights, contact: support@bhub.om
